# こみゃくのもり (Co-MYAKU Forest)

「こみゃくのもり」は、2025年日本国際博覧会（大阪・関西万博）の「EXPO 2025 Design System」からインスパイアされた、非公式のインタラクティブ・アンビエント・ファンアート（二次創作）です。

---

## 概要

画面をタップ（またはクリック）すると、音の粒と共に「こみゃく」たちが森に現れます。  
放置すれば、こみゃくたちが自ら集まり、声を響かせながら静かな森を作り上げていきます。  
日々の喧騒を離れ、こみゃくの森の音に耳を傾けてみてください。

---

## 特徴

- **生成音楽エンジン** — Web Audio API を使用し、アルペジオ・合唱声・鳥のさえずりをリアルタイムに生成します。
- **インタラクティブ** — 画面をタップ／クリックした場所に「こみゃく」が現れ、声を響かせます。
- **ヌシ** — ごく稀に巨大なヌシが現れ、低音の重厚な和声を響かせます。
- **パーソナライズ** — お手持ちの画像を背景としてアップロードし、自分だけの森を作ることができます。HDR写真にも対応しています。
- **PWA 対応** — ホーム画面に追加することで、アプリのように使用できます。

---

## 遊び方

1. **START** ボタンを押して森に入ります。
2. 放置すると自動的に音楽が奏でられます。
3. 画面をタップ（クリック）して、好きな場所にこみゃくを呼び出してみてください。
4. 画面下部のドックから再生・一時停止ができます。
5. ⚙️ 設定アイコンから以下の操作が可能です。
   - 音量の調節
   - テンポの切り替え（120 / 60）
   - チャンネルごとのミュート
   - 背景画像のアップロード・リセット
