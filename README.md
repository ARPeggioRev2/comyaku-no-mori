# こみゃくのもり (Co-MYAKU Forest)

「こみゃくのもり」は、生成される音響と視覚表現が混ざり合う、インタラクティブ・アンビエント・コンテンツです。

## 概要
画面をタップ、または放置することで、分散和音と共に「こみゃく」たちが森に現れます。
日々の喧騒を離れ、静かな森の音に耳を傾けてみてください。

## 特徴
- **生成音楽エンジン:** Web Audio APIを使用し、リアルタイムに分散和音と鳥のさえずりを生成します。
- **インタラクティブ:** 画面をタップすることで、好きな場所に音と「こみゃく」を呼び出せます。
- **パーソナライズ:** お手持ちの画像を背景としてアップロードし、自分だけの森を作ることができます。

## 遊び方
1. `START` ボタンを押して森に入ります。
2. 放置すると自動的に音楽が奏でられます。
3. 画面をタップ（クリック）して、音を響かせてみてください。
4. 右上の設定アイコンから、音量の調節や背景画像の変更が可能です。

## 技術スタック
- HTML5 / CSS3
- JavaScript (Vanilla JS)
- Web Audio API (リアルタイム音声合成)
- Canvas API (ビジュアルレンダリング)
